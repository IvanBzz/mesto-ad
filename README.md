# Mesto

Проект переведён на работу с API Mesto и подготовлен к сборке через Vite.

Перед запуском укажите ваш идентификатор группы и личный токен в `src/scripts/components/api.js`.

Команды:
- `npm install`
- `npm run dev`
- `npm run build`
- публикация через GitHub Actions: `git push origin main`

Ссылка на опубликованный проект:
- `https://IvanBzz.github.io/mesto-production/`
